Микроконтроллер
Микроконтроллер — это электронная микросхема, которая содержит в себе все необходимые компоненты для работы полноценной вычислительной системы:
Популярные семейства микроконтроллеров:
Arduino — для обучения и хобби-проектов

STM32 — для профессиональных устройств

ESP32 — с Wi-Fi и Bluetooth

Raspberry Pi Pico — современные недорогие модели


# Реферат
Современная электроника немыслима без микроконтроллеров — специализированных микропроцессорных систем, предназначенных для управления различными устройствами. Изучение микроконтроллеров является важной частью подготовки инженеров-электронщиков и программистов. В данном реферате рассматриваются типовые среды программирования, базовые модели микроконтроллеров для обучения и подробно анализируется семейство STM32.

# Типовые среды программирования микроконтроллеров для обучения
## 1. Arduino IDE — классика для начинающих
Основные характеристики:
- Бесплатная и открытая

- Кроссплатформенная (Windows, macOS, Linux)

- Простой интерфейс — минималистичный и понятный
  
## 2. PlatformIO — современная альтернатива
Преимущества:
- ✅ Интеграция с VSCode — современный редактор

- ✅ Менеджер библиотек — установка в один клик

- ✅ Поддержка 1000+ плат — от Arduino до профессиональных MCU

- ✅ Встроенная отладка

- ✅ Система сборки — не требует ручной настройки

  ## 3. MPLAB X IDE — для микроконтроллеров PIC
  Особенности:
Официальная среда от Microchip

Поддержка всего семейства PIC

Интеграция с MPLAB XC компиляторами

Компоненты:
MPLAB X IDE — основная среда

MPLAB XC Compilers — компиляторы (бесплатные с ограничениями)

MPLAB Code Configurator — визуальная настройка

## 4. STM32CubeIDE — для ARM Cortex-M
Уникальные возможности:
Интеграция с STM32CubeMX — визуальная конфигурация

Автогенерация кода инициализации

Поддержка HAL (Hardware Abstraction Layer)
